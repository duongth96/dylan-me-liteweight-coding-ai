# liteweight-coding-ai

A lightweight coding AI extension for VS Code.

## Features

- **Hello World Command**: Displays a friendly greeting.

## Requirements

- VS Code 1.80.0 or higher.

## Extension Settings

This extension contributes the following settings:

* `liteweight-coding-ai.enable`: Enable/disable this extension.
