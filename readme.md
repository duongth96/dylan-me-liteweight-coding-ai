# Liteweight Coding AI

## Get Started

- Run ollama from ollama folder, it provides the docker compose file
- Run the extension from vscode, it provides the launch.json file

## Features

- Code completion
- Code explanation
- Code refactoring
- Code generation

## Deployment

- The ollama model is deployed on the ollama server
- The extension is deployed on the vscode marketplace or export file vsix

```bash
# Install vsce if not installed
npm install -g vsce
```

```bash
# Export the extension
vsce package
```
